# 创建一个 Spring Boot 项目

## 打开 Spring Initializr 网页

[Spring Initializr](https://start.spring.io)

## 配置项目信息
- Project: Maven
- Language: Java
- Spring Boot: 3.5.x
- Project Metadata :
    - Group: com.gnerv
    - Artifact: demo01
    - Name: demo01
    - Description: Demo project for Spring Boot
    - Package name: com.gnerv.demo01
    - Packaging: jar
    - Configuration: Properties
    - Java: 21
- Dependencies: 
    - Spring Boot DevTools
    - Lombok
    - Spring Configuration Processor
    - Spring Web
