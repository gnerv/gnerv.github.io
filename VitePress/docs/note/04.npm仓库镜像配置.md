# npm仓库镜像配置

## 使用阿里巴巴仓库镜像

```shell
# 全局配置
npm config set registry=https://registry.npmmirror.com

# 直接使用
npm install --registry=https://registry.npmmirror.com

# electron 镜像
export ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/
```
> electron 镜像用于下载electron相关的包时使用
