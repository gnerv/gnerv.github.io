# 部署Jdk开发环境

## JDK 下载

这里我们使用 [Eclipse Temurin Jdk](https://adoptium.net/zh-CN/temurin/releases?version=25&mode=filter&os=any&arch=any), 选择需要的版本进行下载。


## 解压部署

将下载的部署文件解压到合适的目录下。

## 配置系统环境变量
### Windows

1. 打开环境变量配置页面，新建 JAVA_HOME 环境变量，值为 JDK 部署文件的根目录。
2. 修改 PATH 环境变量，在变量值的开头增加 “%JAVA_HOME%\bin;”
3. 点击保存按钮，自动生效。

### Mac / Linux

1. 打开控制台，执行命令。
```shell
# Mac
vi .zprofile
# Linux
vi /etc/prefile

export JAVA_HOME=${替换为JDK 部署文件的根目录}
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

```
2. 保存修改内容并退出编辑文件
3. 执行命令，使配置生效。
```shell
# Mac
source .zprofile
# Linux
source /etc/prefile
```
4. 执行检查
```shell
java -version

# 控制台输出JDK版本信息表示成功。
```
