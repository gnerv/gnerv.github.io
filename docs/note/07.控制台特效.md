# 控制台特效

适用于Mac/Liunx

## 下载安装文件 [no-more-secrets-master.zip](../public/nms/no-more-secrets-master.zip)

下载完成后解压到合适目录。

## 编译安装

```shell
cd ./no-more-secrets
make nms
make sneakers
sudo make install
```
## 适用方式

在控制台命令执行命令时，增加'｜ nms '

```shell
ls -l | nms
```

## 在环境变量配置文件中预定义

### Mac ～/.zprofile | Linux /etc/profile

```shell
alias ll="ls -l | nms -a -s"
# 保存退出
```

### 使环境变量配置文件生效
```shell
# Mac
source ～/.zprofile
# Linux
source /etc/profile
```
