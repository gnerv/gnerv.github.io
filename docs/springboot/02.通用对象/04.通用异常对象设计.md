# 通用DTO对象设计

## 参数异常

> 用于请求参数检查异常响应

```java
/**
 * <p>
 * 请求参数异常
 * </p>
 *
 * @author Gnerv LiGen
 */
public class ParameterException extends RuntimeException {

    public ParameterException(String message) {
        super(message);
    }

}
```
## 业务异常

> 用于业务代码执行异常响应

```java
/**
 * <p>
 * 业务处理异常
 * </p>
 *
 * @author Gnerv LiGen
 */
public class BusinessException extends BaseException {

    public BusinessException(String message) {
        super(message);
    }

}
```
